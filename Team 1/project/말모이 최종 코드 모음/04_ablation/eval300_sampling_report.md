# eval300 샘플링 설계 보고서

## 목적
baseline / few-shot / v4 3-way ablation을 위한 공정하고 대표성 있는 300개 eval set 구성

## 모집단
- **eval_set_1925.json**: 62,476개 Merged_Corpus_Final(인조 전체)에서 연구팀이 설계한 대표 샘플 1,925개
- 1,925개는 길이·패턴·시기를 반영해 설계된 평가 세트이므로, 이를 모집단으로 사용

## 샘플링 방식
- **방법**: 길이 버킷(XS/S/M/L) × 패턴(GEN/REP/MEM/DEN/APT/ROY/APR) 교차 셀 비례 샘플링
- **목표**: 300개 (1,925의 약 15.6%)
- **시드**: random.seed(42) (재현 가능)

## 결과 분포

| 길이 | 기준 | n | 비율 |
|------|------|---|------|
| XS (<50자) | 짧은 일지 | 133 | 44.3% |
| S (50–149자) | 단문 | 104 | 34.7% |
| M (150–349자) | 중문 | 43 | 14.3% |
| L (350자+) | 장문 | 20 | 6.7% |

| 패턴 | n | 비율 |
|------|---|------|
| GEN (일반) | 142 | 47.3% |
| REP (아뢰기, 啓曰) | 135 | 45.0% |
| DEN (불허, 不許) | 9 | 3.0% |
| MEM (장계/서계, 狀啓) | 7 | 2.3% |
| ROY (전교, 傳敎曰) | 3 | 1.0% |
| APT (제수, 除授) | 2 | 0.7% |
| APR (윤허, 允許) | 2 | 0.7% |

**1,925 원본 대비 KL divergence**: 길이 ≈ 0.000, 패턴 ≈ 0.001 (거의 완벽 반영)
